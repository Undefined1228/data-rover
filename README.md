# DataRover

VS Code에서 PostgreSQL · MySQL 데이터베이스를 탐색하고 관리하는 확장 프로그램.

---

## 설치

### Marketplace
VS Code Extensions 패널에서 `DataRover` 검색 후 설치.

### VSIX 직접 설치
```bash
code --install-extension data-rover-x.x.x.vsix
```
또는 Extensions 패널 → `...` → **Install from VSIX**.
